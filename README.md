### Titre du projet

Notre jeu s'appelle "Python Adventure" c'est un jeu de plateform 3d éducatif conçu pour initier les débutants à la programmation fonctionnelle.

### Pour commencer

Vous devrez executé le fichier python_adventure.exe pour lancer le jeu et y jouer.

### Pré-requis

Ce qu'il est requis pour commencer :
    - système d'exploitation windows 10/11
    - seulement le dossier source

### Installation

Les étapes pour installer notre jeu sont :  
    - Ouvrir le dossier "source"
    - Téléchargé le fichier "Python Adventure.zip" (s'assurer d'avoir au moins 800 Mo de libre sur votre ordinateur)
    - Appuyer sur "Extraire tout" lorsque vous ouvrez le dossier .zip que vous venez de télécharger

### Démarrage

Ouvrir le dossier extrait précédement (le dossier "Python Adventure") et executer le fichier "Python_Adventure.exe"
Maintenant vous pouvez jouer à notre jeu !

### Fabriqué avec

Ce projet a été principalement codé sur Unreal Engine 5. Le language de programmation utilisé est le Blueprint, un système de programmation visuelle basé sur des nœuds. Contrairement à Python, qui repose sur du code écrit, Blueprint permet une approche plus intuitive et accessible à tous, notamment aux débutants. Je tient réellement à souligner que ce choix répond strictement à la consigne interdisant l’usage du C++.
De plus, pour la modélisation des objets/personnages et des animations, nous avons utilisé le logiciel Blender.
